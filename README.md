# GELATI
Guide-signal Extraction Logic &amp; Auto Transform Interface


# Installation
### For Windows
```
git clone https://github.com/nebula-c/GELATI
cd ./GELATI
window_build.bat
```

### For Mac
```
git clone https://github.com/nebula-c/GELATI
cd ./GELATI
./mac_build.sh
```

After the build process, only the generated executable file is needed to run the program.
This file alone can be used on other machines running the same OS, even if they donâ€™t have Python or the required libraries installed.
